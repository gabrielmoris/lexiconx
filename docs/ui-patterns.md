# UI & Styling

## Framework
- **Styling**: Tailwind CSS 4 with `@tailwindcss/postcss`
- **Components**: Custom reusable components in `components/`
- **Animations**: Framer Motion for transitions

## Tailwind Usage
- Use utility classes for one-off styles
- Extract repeated patterns into components
- Use CSS variables for theme values

## Component Patterns
- Export default for pages, named exports for components
- Keep components focused and single-purpose
- Use composition over configuration (children props)

## Accessibility
- Use semantic HTML elements
- Include `aria-*` attributes for interactive elements
- Ensure keyboard navigation support
